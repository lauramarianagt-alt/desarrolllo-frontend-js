# Wallafront

Aplicación frontend inspirada en Wallapop desarrollada con JavaScript y Sparrest.js como backend REST.

## Funcionalidades

### Autenticación
- Registro de usuarios
- Login con JWT
- Logout
- Rutas protegidas

### Gestión de anuncios
- Listado de anuncios
- Detalle de anuncio
- Crear anuncios
- Editar anuncios
- Eliminar anuncios
- Validación de ownership

### Estados UI
- Loading
- Error
- Empty
- Success

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- Sparrest.js
- JWT Authentication
- Fetch API

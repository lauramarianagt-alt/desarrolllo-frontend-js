# Wallafront

Aplicación frontend inspirada en Wallapop desarrollada con JavaScript y Sparrest.js como backend REST.

Repositorio GitHub:
https://github.com/lauramarianagt-alt/desarrolllo-frontend-js

Versión publicada con GitHub Pages:
https://lauramarianagt-alt.github.io/desarrolllo-frontend-js/

Para que las funcionalidades completas fruncionen correctamente, es necesario levantar sparrest.js en localhost:8000.

## Funcionalidades

### Autenticación
- Registro de usuarios
- Login con JWT
- Logout
- Rutas protegidas

### Gestión de anuncios
- Listado de anuncios
- Detalle de anuncio
- Crear anuncios
- Editar anuncios
- Eliminar anuncios
- Validación de ownership

### Estados UI
- Loading
- Error
- Empty
- Success

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- Sparrest.js
- JWT Authentication
- Fetch API
